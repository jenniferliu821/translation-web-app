## 步骤 2️⃣：在 Railway 部署（3分钟）

### 2.1 登录 Railway
1. 访问：https://railway.app
2. 点击 "Login with GitHub"
3. 授权登录

### 2.2 创建项目
1. 点击 "New Project"
2. 选择 "Deploy from GitHub repo"
3. 搜索并选择 `translation-web-app`

### 2.3 设置 API Key（重要！）
1. 点击您的项目
2. 点击 "Variables" 标签
3. 点击 "New Variable"
4. 输入：
   - **Name**: `GOOGLE_API_KEY`
   - **Value**: `AIzaSyCp5TKySowNOh20kOZ4ge5N4S6QSjANa0s`
5. 点击 "Add"

### 2.4 生成公开网址
1. 点击 "Settings" 标签
2. 找到 "Networking" 部分
3. 点击 "Generate Domain"
4. 等待几秒钟，会生成类似这样的网址：
   ```
   https://translation-web-app-production.up.railway.app
   ```
