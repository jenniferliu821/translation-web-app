### 错误信息可能包含：
```

或
google.api_core.exceptions.PermissionDenied
```

### 解决方案：
✅ **添加环境变量**

1. 点击 **Variables** 标签
2. 点击 **New Variable** 或 **Add Variable**
3. 输入：
   - **Name**: `GOOGLE_API_KEY`
   - **Value**: `AIzaSyCp5TKySowNOh20kOZ4ge5N4S6QSjANa0s`
4. 点击 **Add** 保存
5. Railway 会自动触发重新部署

## 🔧 常见错误 2：依赖安装失败

### 错误信息可能包含：
```
ERROR: Could not find a version that satisfies the requirement
或
pip install failed
```

### 解决方案：
需要修改 requirements.txt

---

## 🔧 常见错误 3：端口配置问题

### 错误信息可能包含：
```
Port 8080 is already in use
或
Application failed to respond
```

### 解决方案：
检查 Procfile 配置（应该已经正确）

---

## 🔧 常见错误 4：Python 版本问题

### 错误信息可能包含：
```
Python version not supported
或
Incompatible Python version
```

### 解决方案：
添加或修改 runtime.txt
